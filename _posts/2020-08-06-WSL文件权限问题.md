最近在 WSL 下碰到 `PHP chmod(): Operation not permitted` 这个问题，
看起来是文件权限的问题，
不过查看该文件的权限是 777，
不知道是 Bug 还是什么原因导致的，

找到一篇文章是关于这个, 原因出资Wsl的挂载
解决办法是更改 wsl.conf 文件。

```shell script
sudo vim /etc/wsl.conf
```

```apacheconfig
[automount]
enabled = true
options = "metadata,umask=22,fmask=11"
mountFsTab = false
```
